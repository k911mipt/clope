usage: Clope -d <database_file> -r <repulsion> [-o <output_file>]
Исходный файл должен содержать в себе список транзакций.
У каждой транзакции атрибуты должны быть разделены запятыми
объектом считается слово между запятыми
объекты с одинаковым названием, но разными номерами атрибутов считаются разными

первый атрибут транзакции является её классом

Программа выполнит кластеризацию алгоритмом Clope и выведет на экран разбиение классов по кластерам после инициализации и после окончания работы
В случае указания вывода в текстовый файл, программа выведет номера кластеров, разделённые переносом строки
Каждая строка в сформированном файле будет соответствовать строке транзакции в исходном.
